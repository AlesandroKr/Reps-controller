# **README — Contador de Repeticiones (4 Fases + Series + Descanso)**

## **Descripción**

Aplicación web que permite controlar repeticiones físicas con un temporizador dividido en 4 fases:

- Pausa 1
    
- Bajada
    
- Pausa 2
    
- Subida
    

Incluye:

- Ajuste de tiempo para cada fase
    
- Selección de número de series y repeticiones
    
- Descanso automático entre series
    
- Sonidos para cada transición
    

---

## **Características**

- Reproducción automática de las fases
    
- Conteo de repeticiones y series restante
    
- Posibilidad de pausar y reiniciar
    
- Configuración personalizada de tiempos
    
- Totalmente funcional en escritorio y móvil
    
- Sonidos diferenciados para cada fase
    

---

## **Tecnologías utilizadas**

- HTML
    
- CSS
    
- JavaScript
    

---

## **Cómo usarlo**

1. Selecciona el número de series y repeticiones.
    
2. Ajusta los tiempos.
    
3. Presiona **Start**.
    
4. Descanso entre series automático según el tiempo configurado.
    
5. Puedes pausar o reiniciar en cualquier momento.
